# 🤖 Discord Multipurpose Bot

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Discord.py](https://img.shields.io/badge/discord.py-2.0+-green.svg)](https://discordpy.readthedocs.io/)
[![License](https://img.shields.io/badge/license-MIT-purple.svg)](LICENSE)

Bot Discord dengan fitur Music, Welcome System, dan RPG text

## ✨ Fitur

### 🎵 Music System
- Play musik dari YouTube
- Queue management dengan shuffle & loop
- Volume control (0-200%)
- Now playing display
- Auto leave saat sendiri

### 👋 Welcome System
- Custom welcome messages dengan image card
- Goodbye messages
- Variables: {member}, {server}, {member_count}
- Mudah dikonfigurasi per server

### ⚔️ RPG Game
- 4 Class: Warrior, Mage, Archer, Healer
- Monster hunting dengan level scaling
- Epic boss battles
- Shop & inventory system
- Equipment system
- Daily rewards
- Player vs Player duel
- Leaderboard system
